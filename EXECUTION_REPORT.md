## 🎯 项目核心功能介绍

**EnvPilot-CLI** 是一款专为开发者打造的轻量级终端环境变量智能管理引擎。

### 解决的痛点

在现代软件开发中，开发者经常面临以下问题：
1. **多项目切换困难** - 每个项目都有不同的API密钥、数据库连接等环境变量
2. **.env文件管理混乱** - 手动编辑容易出错，且难以追踪变更
3. **敏感信息泄露风险** - 不小心将密钥提交到版本控制
4. **环境切换效率低** - 在开发、测试、生产环境间切换需要反复修改配置

### 自研差异化亮点

| 亮点 | 说明 |
|------|------|
| **🎯 零依赖设计** | 仅使用Python标准库，单文件可直接运行，无需pip安装任何依赖 |
| **📁 多配置隔离** | 支持无限数量的项目配置，每个配置完全独立 |
| **🔐 智能敏感保护** | 自动识别包含key/secret/password/token/api的变量并隐藏 |
| **⚡ 一键激活** | 快速导出.env文件，配合source命令即时生效 |
| **📥 无缝迁移** | 自动解析现有.env文件，零成本迁移现有项目 |
| **🎨 彩色TUI界面** | 美观的终端输出，提升命令行使用体验 |

## ⚡ 快速启动命令

```bash
# 方式一：直接下载使用
curl -o envpilot.py https://raw.githubusercontent.com/gitstq/EnvPilot-CLI/main/envpilot.py
python3 envpilot.py --version

# 方式二：pip安装
pip install envpilot-cli

# 快速开始
envpilot init myproject --desc "我的项目"
envpilot add myproject API_KEY "sk-your-api-key"
envpilot activate myproject
source .env
```
